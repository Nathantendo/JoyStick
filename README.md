# JoyStick
A Simple Launcher App For Files, Apps, And Websites.

With JoyStick, You Can Launch Your Content, Your Way.

JoyStick Is In Active Development, So Expect Missing Features And Bugs.

# Intended Uses
JoyStick Is Intended For Games, But It Can Be Used For So Much More!
